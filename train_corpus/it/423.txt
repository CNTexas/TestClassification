

　　美研究称女性患者4年内心脏病发作几率达到10%，其发病率高于男性
 ; ; ; ;因心绞痛而初次前往医院进行检查时，美国女性金・卡切曼・格尔兹被医生认定未有任何问题。两年后，格尔兹才被检测出心脏供血不足，进而确诊为微血管病变。美国雪松－西奈山医学中心研究显示，患有微血管病变的女性在4年内心脏病发作几率达10%，远远超出普通心脏疾病的死亡率。另外，微血管病变在女性中的发病率高于男性。 
 

心脏病专家正就女性心脏患病进行研究。


手术医生正为一名女性患者进行心脏搭桥手术。


 ; ; ; ;病例 一“健康”女性突发心绞痛 
 ; ; ; ;现年39岁的金・卡切曼・格尔兹生有三个孩子，她一直崇尚健康的生活方式：每天慢跑5英里，吃麦片早餐，从不抽烟。格尔兹从未超重，血压正常；据她自己说，家族遗传非常健康，其曾祖母高寿至102岁，“我一直认为，像我这样健康的人最不可能患上心脏病。” 
 ; ; ; ;“风云难测”，自2003年开始，格尔兹突然出现心绞痛、胸痛等症状。经全面检查后发现，这是由于血液不规律流入心脏引起的。此外，格尔兹的一个心室出现肿大症状，心脏瓣膜无法正常“运行”。 
 ; ; ; ;为了控制病情，格尔兹每天要服用4种药物，今后很可能还要接受更多治疗。服药不可能解决所有问题，胸痛使其无法坚持跑步，格尔兹只得用散步和瑜伽来延续自己所崇尚的健康生活方式。 
 ; ; ; ;生活并未被突如其来的病痛给彻底打乱，格尔兹还是很乐观，她说：“这并非被判死刑，我的下半生不会因此抑郁。”但是金的心脏病究竟会发展到什么地步，现在还不得而知。目前的诊断并不悲观，但这一病痛很可能缩短她的寿命。 
　　研究 女性微血管易病变 
 ; ; ; ;雪松-西奈山医学中心的墨尔兹博士目前正在领导一项美国政府资助的妇女心脏血管局部缺血评估。她透露，目前美国大约有3百万妇女患有微血管病变。最新研究显示，患有微血管病变女性在4年内心脏病发作几率达到10%，远远超出普通心脏疾病的死亡率。此外，微血管病变在女性中的发病率高于男性，但原因仍在研究之中。 
 ; ; ; ;虽然微血管病变的危害巨大，但检测出来并不容易。 
 ; ; ; ;按照传统的诊断方式，当医生未能从血管造影中发现异样时，他们通常不会推荐病人接受更多检测。当格尔兹由于心绞痛前去求医时，最初的检测没有发现任何问题。 
 ; ; ; ;格尔兹在怀第三个孩子时第一次出现了微血管病变的症状。这导致她的心脏出现了严重功能障碍，差点在生产时死去。生下第三个孩子后，格尔兹经常出现胸痛，并在半夜突发心脏病。直到两年后，格尔兹才被检测出心脏供血不足，进而确诊为微血管病变。 
 ; ; ; ;美国国家心脏研究中心的乔治・索普克博士呼吁更多医生不能再仅仅依靠血管造影来诊断，而是用全面的检测来确保没有误诊。他说，当一名女性患者因为心脏问题来就诊时，应当进行全面的测试，考虑所有的因素，包括心理测试。 
 ; ; ; ;比较 心血管病成女性杀手 
 ; ; ; ;研究还显示：目前，心血管疾病已成为美国年龄在25岁以上女性的头号健康杀手，每年被心血管疾病夺去生命的25岁以上女性超过25万人。从1984年至今，每年因心脏病死亡的女性人数均超过男性。由于医疗技术的进步，心血管疾病死亡率有显著下降，然而大多数状况的改善都发生在男性患者身上。 
 ; ; ; ;对出现胸痛等其他心脏病症状的患者进行检查时，女性比男性的心血管状况更好，出现血栓的几率小，这表明还有其他原因导致女性胸痛。患有冠状动脉血栓的女性普遍比患同类疾病的男性年龄大，而且症状更加严重。 
 ; ; ; ;这些妇女通常也会出现其他健康问题，如高血压和高血糖等，这导致实施心脏手术的难度更大。此外，她们出现心脏衰竭的几率也更大，由于心肌衰弱，这可能导致患者死亡。 
 ; ; ; ;当女性患者接受心脏搭桥手术或清除血栓的手术时，手术的成功几率比男性小，而且她们更容易受到副作用的影响。能准确检测出男性心脏疾病的血液测试对于女性并不完全适用。另外，女性比男性更容易因严重精神压抑引发暂时性心理衰竭。 
　　说法 更多男女差异待研究 
 ; ; ; ;洛杉矶雪松－西奈山医学中心的心血管疾病学家诺埃尔・拜尔雷兹・墨尔兹博士表示：“我们每进行一次研究，就会发现更多基于性别的差异，由此可见研究的重要。” 
 ; ; ; ;波士顿大学心血管学家艾莉斯・雅各布认为，由于女性身型通常比男性小，因此进行心脏手术后康复的情形不如男性；同时她们的血管比男性更细，因此也更容易再次被血栓堵塞。 
 ; ; ; ;此外，进行心脏手术的医生使用的疏通被堵塞心血管的医疗器具大部分适合男性血管，并不适用更细的女性血管。现在，医生们懂得了使用适合女性血管的器具，手术的成功率也在上升。不过，男女两性患心脏病的差异的根本原因还是没有合理的解释。
