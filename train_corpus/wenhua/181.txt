 ; ; ; ; ; ; ; 受人之托，特意到书城找一本刚出版的新书，未曾想，花了近两个小时也没有找到。在我们的感召下，几位热心的楼层管理员也加入了寻书行动，最终还是一无所获。 ; ; ; ; ; ; ; ;到了不得不空手而归时才猛然醒悟，那么大一个书城，偶然碰到一百本书也不希奇，要有目标地寻索一本却谈何容易！即便是知道了它的名字，在电脑中也看到了它的踪影，它明明就在这书的密林之中，要找到却也不易。 ; ; ; ; ; ; ; ;这让人想起博尔赫斯的《沙之书》里的一句话：隐藏一片树叶的最好地点是树林，进而让人想起嘈杂喧嚣的人群……有一个词立时浮现在眼前――茫茫人海。是的，人一旦陷入这人海，就等于一粒沙陷入沙漠，一滴水跌入大海。沙漠有吞噬绿洲的力量，大海有冲决堤岸的气势，人潮也有摧毁一切秩序的潜能。而一粒沙却找不到自己，每一粒沙都找不到自己；一滴水找不到自己，每一滴水都找不到自己；一个人找不到自己，每一个人都找不到自己。 ; ; ; ; ; ; ; 他们因为众多而力量强大，他们因为众多也变得更加渺小――个体因为同类数量的增加而互相壮大，也因为同类数量的增加而互相遮蔽。一粒沙的生命因为沙堆而延续，却因为沙海而丧失；一滴水的生命因为江河而延续，却又死于大海；一个人的生命价值因为群体而得以彰显，却又淹没于芸芸众生，甚至因互相践踏而殒命。 ; ; ; ; ; ;  ;法国社会心理学家古斯塔夫・勒庞把那些丧失个性或被某种意识形态感染而聚合在一起的人群称为“乌合之众”。“构成这个群体的个人不管是谁，他们的生活方式、职业、性格或智力不管相同还是不同，他们变成了一个群体这个事实，便使他们获得了一种集体心理，这使他们的感情、思想和行为变得与他们单独一人时的感情、思想和行为颇为不同。若不是形成了一个群体，有些闪念或感情在个人身上根本就不会产生，或不可能变成行动。心理群体是一个由异质成分组成的暂时现象，当他们结合在一起时，就像因为结合成一种新的存在而构成一个生命体的细胞一样，会表现出一些特点，它们与单个细胞所具有的特点大不相同。”个体的意识个性淹没在群众心理之中，群众心理诱发出情绪，意识形态通过情绪感染得到传播。一旦被广泛传播，意识形态就渗透到群众中个体的层次，使个体丧失批判能力……这是一种心理上的匿名状态――这是孤独软弱的个体欲表现其力量时必须借助的途径，也是个体表现短暂的虚假强大时必须付出的代价――他们在人群中将彻底丧失自我意识、身份乃至生命。 ; ; ; ; ;  ; 容易被人们忽视的是，在盲目涌动的人群――即所谓乌合之众中，个体的人同样是失去关系庇护的个体，在嘈杂拥挤的人群中，人更能体会到一种孤独和无助。所谓“大隐隐于朝，中隐隐于市”，隐，就是一种互相掩护，互为庇荫；而真正要实现隐的目的，就是埋没自己的才华，磨灭自己的个性，最终以彻底丧失自我而告终。 ; ; ; ; ; ; ; 皇帝之所以为皇帝，总需要前呼后拥的一帮护卫和随从来证明；实在没有这些活的背景，器物和服饰也是身份的象征。所谓君子不重则不威，是说器物和排场不隆重，服饰不繁复，就不足以体现威严和威风。受制于这样的重礼重服，对皇帝作为个体的自由当然是一种压抑；无怪乎许多皇帝都有微服出行的冲动。微服即可隐身。微服就是轻装简从，而微服的诱惑力和微服的危险性正在于此。 ; ; ; ; ; ; ; 在明处的皇帝固然容易受到意料之中的攻击，微服私访中的皇帝则难免受到意外的伤害。因为，处于匿名状态的皇帝获得了作为一个自然人的自由，却失去了特殊身份的庇护。因而是刺激的，也是危机四伏的。 ; ; ; ; ; ; ; 书可以是有生命的活物，如果它能独立存在，或有幸被置于读者注目的焦点；书也可以是没有生命的僵尸，如果它不幸卷入书的乌合之众，与众多的同类互相遮蔽乃至践踏致死。这是一本书的命运，一本生于信息爆炸时代的书的宿命。 ; ; ; ; ; ; ; 幸而人略有不同，人永远不会停止与这样的命运相抗争；虽然人也有宿命，但是人，却从来不会就那么轻易地乖乖认命。 ;
