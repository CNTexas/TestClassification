 ; ; ; 汉诺威交通概览
 ; ; ; 您只要跟著行人路上的红线指示走，便可游览市内各主要名胜。从火车站前的游客咨询处出发到返回原处，全程需时2至3小时。咨询处内可索取[红线区](Red Thread)小册子（英文版）参阅。>>>汉诺威旅行小贴士 ; ; ; 交通指南
 ; ; ; 汉诺威地处德国北部地区的交通要塞，距离柏林289公里，汉堡151公里，法兰克福330公里。它位于北德平原和中德山地的相交处，正处于巴黎到莫斯科、北欧到意大利的十字路口，又濒临中德运河，是个水陆辐辏的交通枢纽。 
 ; ; ; 机场：
 ; ; ; Flughafen Hannover机场代号HAJ，距离市区12公里。出租车要20欧元。机场的公共车到火车站要15分钟。详细资料见机场的网站。 
 ; ; ; 火车：
 ; ; ; 汉诺威火车站是德国重要的火车中转站。乘ICE城市高速列车2个半小时可到法兰克福，
 ; ; ; 市内交通 可以乘坐地铁、电车、公共汽车等交通工具
