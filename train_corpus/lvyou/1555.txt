 ; ; ; ; ; ; ; 参观充满浪漫情调的夏洛滕堡宫及宫殿的博物馆，走进中世纪的柏林，体验哈克庭院喧闹的生活感受。
 ; ; ; 柏林三日游 
 ; ; ; ; ; ; ; 三天的时间您可以比较放松地参观景点和博物馆。此外我们还推荐您到城南的王宫或到东城区的一些景点去看看。
