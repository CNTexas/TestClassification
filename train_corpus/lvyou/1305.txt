 ; ; ; 铁路交通状况
 ; ; ; 
从法兰克福乘坐IC特快，约2小时15分钟后，可抵达科隆。这里主要有通往比利时和荷兰方向的国际列车。 ; ; ; 
火车：
 ; ; ; 科隆火车站(Koln 
Hauptbahnhof)：是德国西部重要的中转站。前往比利时、法国的火车大多需要在此中转。科隆火车站有两个出站口西面的出口出站就是科隆大教堂，公共车站在东面的出口外。 ; ; ; 
机场：
 ; ; ; 科隆-波恩机场(Flughafen 
Koln-Bonn)在科隆与波恩中间，是两城市共用的机场。机场代号CGN，距离科隆20公里。乘出租车到市区要20分钟。有火车连接机场到科隆中心火车站和杜塞尔多夫火车站。(www.airport-cgn.de)
