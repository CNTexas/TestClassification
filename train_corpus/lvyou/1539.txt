 ; ; ; 德国交通介绍
 ; ; ; 现在前往德国的直航机只有停靠法兰克福市和慕尼黑市的班机。法兰克福是欧洲的交通中心，从中国飞德国一般以法兰克福机场为终点站，所有的旅程都从法兰克福机场开始。如要飞往德国其它城市，就只有德航转机可选。无论您是在那里转机或转乘火车，需要首先通过入境关卡 (Passport Control) 的检查。持中国护照、香港特区护照或 BNO 之旅客，请于非欧盟国 (NON-EU Nations) 的入境关卡排队办理手续，无须填写入境，但请准备好转接航班的登机卡或机票。 ; ; ; 租车服务
 ; ; ; 须持国际驾照，车辆靠右行驶，城镇里的最高时速为50公里，高速公路原无时速限制，但最近德国政府正在提倡改为最高时速100公里，如欲前往德国开车者，请注意当地的标志。
 ; ; ; 出租车
 ; ; ; DM3.80起跳，每公里DM2.15~2.55，自法兰克福机场到市区约需DM40，约20分钟可抵达。
 ; ; ; 公共巴士交通状况
 ; ; ; 连接城市和偏远地区、车站通常在火车总站附近。同样的距离，乘公交车的费用要比乘火车贵。
 ; ; ; 铁路交通状况
 ; ; ; 德国的火车站称为Bahnhof，而位於大城市的中央火车站则称为Hauptbahnhof（简称Hbf.）。Hbf.大都是各大城市的火车终点站（如法兰克福、慕尼黑等），随着德国统一而产生的德国火车通行证( German Rail Pass.) ，可以自由搭乘DB及DR任何路线的火车。EC（Euro City）和IC（Inter City）只在6：00―10：00运营，每小时一班，疾驶于各大主要城市建。ICE（Inter City Express）列车时速可达280公里。火车总站（der Hauptahnhof）站内告示牌用黄色标志标示出发站（Abfahrt），白色标志标示到达站（Ankunft）。
 ; ; ; 使用火车通行证的说明：
 ; ; ; ●可在所购火车票效期限内，不限次数搭乘德国国铁所有的火车；但票价不含订位、餐饮及睡卧铺。 
 ; ; ; ●使用之天数效期为1个月内任选 4, 5, 6, 7, 8, 9 或 10 天 
 ; ; ; ●分为头等舱或二等舱 
 ; ; ; ●2人同行需全程一起搭乘火车旅行。 
 ; ; ; ●以每2人为一单位开出的票券，如3人同行时，则需开出一张2人同行票及一张个人票；若4人同行则须开2张2人同行票 
 ; ; ; ●青年票：适用于启用通行证搭乘火车第一天仍未满26岁之青年。 
 ; ; ; ●儿童票适用于 6-12岁(未满12岁) 之儿童，为成人票面之半价。 
 ; ; ; ●6岁以下儿童不占位(未满6岁)与大人同行免费。
 ; ; ; 依照当地情况，价格可能会有不同的优惠，请到当地确认。
 ; ; ; 航空交通状况
 ; ; ; 北京、上海等城市有航班直飞德国的慕尼黑和法兰克福等城市，北京飞慕尼黑的航班，参考价：单程票价人民币3200元，往返票价人民币5500元。法兰克福是欧卅的交通中心，从中国飞德国一般以法兰克福机场为终点站，如要飞往德国其它城市，就只有德航转机可选。
