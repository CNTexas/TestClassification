
 ; ; ; 交通指南
 ; ; ; 
纽伦堡地处中欧的地理中心，恰好处于巴黎一布拉格、伦敦一布鲁塞尔一维也纳、斯德哥尔摩一苏黎世一米兰、柏林一罗马等几条高速公路的交汇点上。
 ; ; ; 机场：
 ; ; ; 纽伦堡国际机场，机场每天有50条直达线路通向20几个欧洲的中心地区；
 ; ; ; 火车：
 ; ; ; 
纽伦堡火车站，纽伦堡还与欧洲高速轨道网连接在一起，是铁路网中ICE、EC和IC线路的结合点，特别是连接欧洲东部和东南部的交通枢纽；
 ; ; ; 港口：
 ; ; ; 
纽伦堡港是北巴伐利亚最重要的运输枢纽，每年的总货运量达700万吨；流贯市区的莱茵――多瑙运河从1992年起成为连接莱茵与多瑙、北海与黑海的一个环节。
