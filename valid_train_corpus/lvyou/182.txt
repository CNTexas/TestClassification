

开封公园开始“备战”“五一”黄金周 


重庆千年古镇磁器口准备迎接“五一”黄金周


动物备战黄金周
 ; ; ; 
今天，距离今年中国“五一”旅游黄金周还有三天。据中国假日旅游部际协调会议办公室最新发布的“五一”黄金周旅游信息预报显示：重点旅游城市交通预订趋热，住宿预订升温但尚有空间。
 ; ; ; 统计信息显示，黄金周临近，中国重点旅游城市交通预订趋热，游客主要前往北京、上海、桂林、张家界、香港等热点旅游城市，出行集中在四月三十日至五月一日，回程集中在五月六日至七日；旅游住宿预订比前两天明显升温，五月二日至五日星级饭店预订普遍达到百分之五十，部分城市和景区已经突破百分之六十，家庭旅馆等社会住宿预订达到百分之三十；城市周边旅游住宿预订开始启动。
 ; ; ; ;面对旅游预订“一片大好”的欣欣向荣之景，各地纷纷“摩拳擦掌”，紧张“备战”。其中天津的旅游局高官亲自带队，会同市工商局、物价局、卫生局、交通道路运输管理处等九部门，进一步对塘沽海上旅游服务公司、海河游览管理处、凯悦饭店、三源电力国际旅行社、西青区石家大院等旅游企业进行联合大检查，确保“五一”黄金周旅游市场“安全、有序、健康、繁荣”。 
 ; ; ; ;在东北名城哈尔滨市的太阳岛风景区，“首届太阳岛楹联展” 
“太阳岛?天龙俄罗斯人体艺术摄影展”等系列热点主题活动将陆续推出，作为“太阳岛百年庆典”的开篇序曲；哈尔滨游乐园推出“欢乐‘五一’大行动倾情大酬宾”系列活动。据悉，该园花费近千万元引进的四项东北三省之最的惊险游乐项目，也将在“五一”黄金周期间陆续与游人见面。 

 ; ; ; ;在有着“人间天堂”之美誉的杭州市，结合今年“中国乡村旅游年”，重点打造临安“乡村风情旅游”吸引各地游客。除了欣赏浙西风情演出，游客还有机会被当地聘请为临安乡村的荣誉村民。在享受免费回村“探亲”之余，游客还可参与本村“农家乐”的管理，感受到正宗、地道的浙西乡村田园生活。(完）
