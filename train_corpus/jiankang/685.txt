连续几天的浮尘天气让人没精打采，准备装修的家庭更犯起了嘀咕：污染严重天气干燥的春季，如何能够装修出健康干净的家居空间呢？除了在装修过程中把握环保之外，平日里的生活中也要多加注意。本期我们集纳了可能伴随装修、家具、用品以及办公设备等等溜进家门的六大污染物，并将它们可能侵入的渠道、可能带来的危害以及赶走它们的方法告诉您。




 ; ; ●一号污染物：苯（包含苯、甲苯、二甲苯、苯并芘）
 ; ; 来源和存在：苯系物主要存在于油漆、涂料、乳胶漆等化工产品里以及含有苯的各类装修辅料。
 ; ; 
对健康的危害：苯系物被人体吸入后，可出现中枢神经系统麻醉作用；可抑制人体造血功能，再生障碍性贫血患病率增高；还可导致胎儿的先天性缺陷等。
 ; ; 
国家标准：《民用建筑室内环境污染控制规范》中规定苯含量Ⅰ类和Ⅱ类民用建筑工程均需≤0.09毫克/立方米。
 ; ; 
应对策略：必须选择符合国家标准的装饰材料，特别是油漆、涂料、胶黏剂和防水材料，最好选择水性木器漆。苯系物挥发较快，装修后保持良好通风，并闲置数月后再行入住即可。入住前最好联络专业室内环境检测机构检测苯系物污染程度。

 ; ; ●二号污染物：甲醛
 ; ; 
来源和存在：加入减水剂的水泥墙面通常会加入甲醛，成为甲醛污染的主要来源。家具和橱柜材料中的胶合板、密度纤维板、刨花板……遇热、潮湿时甲醛就会释放出来。不合格的涂料和乳胶漆、黏着剂、织物、地毡等等都是甲醛释放的来源。
 ; ; 
对健康的危害：甲醛和苯是导致白血病的两大重要污染物。长期接触低剂量甲醛可引起慢性呼吸道疾病，引发癌症、白血病，儿童和孕妇对甲醛尤为敏感。
 ; ; 
国家标准：《民用建筑室内环境污染控制规范》中规定甲醛含量为Ⅰ类民用建筑工程需≤0.08毫克/立方米、Ⅱ类民用建筑工程需≤0.12毫克/立方米。
 ; ; 
应对策略：用户可以购买甲醛检测剂粗评室内甲醛污染状况，简单易行。如检测结果可疑，应该立即联络专业机构检测甲醛污染程度，并采取治理措施；装修选料的时候就需要考虑会不会造成甲醛污染，装修后最好通风3个月到半年后再行入住。除此之外，尽可能多开窗通风，如有条件，安装新风机。

 ; ; ●三号污染物：氨气
 ; ; 
来源和存在：氨气大多是从墙体中释放出来的，墙体的面积决定氨的释放量，不同房间空气中氨污染的程度也不同。
 ; ; 
对健康的危害：氨气可减弱人体对疾病的抵抗力。氨被吸入肺后破坏运氧功能。短期内吸入大量氨气后可引发各种不适，严重者可发生肺水肿、成人呼吸窘迫综合征。
 ; ; 国家标准：《室内空气质量标准》规定氨浓度需≤0.2毫克/立方米。
 ; ; 
应对策略：必须通过专业室内环境检测机构检测氨气污染程度，采取治理措施，除此之外，尽可能多开窗通风，如有条件，安装新风机。

 ; ; ●四号污染物：可吸入颗粒物（主要成分是二氧化硫和二氧化氮）
 ; ; 
来源和存在：可吸入颗粒主要来自于生活废气、工业废气和汽车尾气，可吸入的颗粒物非常细小，可以轻易穿越密闭性很好的门窗进入室内。冬天的燃煤以及吸烟会产生大量的可吸入颗粒物。植物孢子、毛发碎屑、粉尘、细菌……均容易和可吸入颗粒物结合。 

 ; ; 对健康的危害：可吸入颗粒经呼吸道进入体内，引发慢性鼻咽炎、慢性气管炎等，并可能致癌。
 ; ; 国家标准：《室内空气质量标准》规定可吸入颗粒物浓度需≤0.15毫克/立方米。
 ; ; 
应对策略：大风或扬沙天，室内空气中的可吸入颗粒物会明显增加，逢此天气要紧闭门窗，增加室内湿度可以有效抑制空气中的可吸入颗粒，比如养鱼、经常拖地、购买加湿器等，植物也是可吸入颗粒的克星。
 ; ; ●五号污染物：TVOC（总挥发性有机物）
 ; ; 
来源和存在：TVOC是指可以在空气中挥发的八类有机化合物的总称，室内的TVOC主要存在于建筑材料中的人造板、泡沫隔热材料、塑料板材；室内装饰材料中的涂料、油漆、黏合剂、壁纸；办公用品中的油墨、复印机、打字机等。
 ; ; 
对健康的危害：医学专家研究表明暴露在高浓度TVOC污染的环境中，可导致人体的肝、肾和血液中毒，在低浓度下会引发各类不适症状。
 ; ; 
国家标准：《民用建筑室内环境污染控制规范》中规定TVOC含量为Ⅰ类民用建筑工程需≤0.5毫克/立方米、Ⅱ类民用建筑工程需≤0.6毫克/立方米。
 ; ; 
应对策略：装修后，TVOC是否超标，需要请专业室内空气检测机构予以检测。并且一定要让房屋空置数月再入住，使总挥发性有机物比较彻底地散发，入住后保持通风，或购买新风机。
 ; ; ●六号污染物：氡
 ; ; 
来源和存在：装修房子时选用的一些石材，比如花岗石以及由石材构成的陶瓷、带有釉面的工艺品等等。
 ; ; 对健康的危害:氡气具有对人体健康不利的放射性，特别是容易导致肺癌。
 ; ; 国家标准：不得大于400Bqm3。
 ; ; 
应对策略：需要专业检测机构协助检测室内氡辐射是否超标，家装时应尽可能少用石材，尤其是花岗石。马桶等陶瓷洁具产品要注意选择合格的产品。
