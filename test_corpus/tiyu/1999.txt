在45岁生日的第二天，现任米德尔斯堡主帅收到了一份迟到的生日礼物！前天晚上英足总正式宣布，在经过两个多月的遴选之后，麦克拉伦最终成为埃里克森的继任者。在斯科拉里退出的情况下，麦克拉伦从“最佳备选”晋升为“最终候选”，而他也是在击败阿拉代斯、科比什利和奥尼尔这三位竞争者后顺利上位的！已经在国家队辅佐埃里克森6年的麦克拉伦立即变为全英焦点，尽管要到8月1日才正式上任，但麦克拉伦已经成了媒体轰炸的对象，而这仅仅是个开始！ ; ; 

　 ; ; 最佳礼物乐坏麦克拉伦 ; 
　 ; 
“这是我职业生涯的最高点，我将全力迎接这一巨大挑战。”当地时间周四下午，麦克拉伦在英足总为他召开的发布会上侃侃而谈！ ; ; 
　　 
与阿拉代斯、科比什利和奥尼尔一样，作为英足总起先的候选者，麦克拉伦并没有过人的本事。他带领米德尔斯堡的最佳成绩是联赛第七，他作为独立主帅的惟一荣誉是联赛杯冠军，除此之外他可以炫耀的似乎只剩下作为曼联助理教练在1999年夺得三冠王！刚过45岁的麦克拉伦论名气比不上斯科拉里，论大赛经验比不上奥尼尔，但是在国家队担任助教的这6年为他作了最好的铺垫―――足总上下人头熟络，国脚中的口碑也相当不错。当然，运气也很帮忙，斯科拉里的意外拒绝和足总派系的争执不下最终让麦克拉伦顺理成章地获胜，因为他是最安全最稳妥的选择！ ; ; ; 

　　麦克拉伦本不是足总的第一选择，但是被逼到眼红的英足总只能狠狠心与米堡主帅签下了合同。下周，麦克拉伦将接到足总的聘书，并得到250万英镑的年薪。由于在今年1月麦克拉伦刚和米堡俱乐部续约了三年，足总将为这次任命付给米堡175万英镑的违约金。对于是否是第一选择，麦克拉伦似乎管不了那么多了：“是不是第一选择并不重要，就像体育比赛时你有时会是第一，有时会是第二第三，重要的是现在我是英格兰国家队的主教练。”不过尽管麦克拉伦无法让足总每个人都满意，但他显然是几个备选中最有资格当选的，“我随英格兰工作五年之久，这里的情况我很熟悉，我知道届时将面对怎样的压力，不管面前有怎样的挑战，我都会作出良好表现。” ; ; ; 

　　麦克拉伦上任后最大的挑战是什么？不是成绩也不是球队更衣室气氛，而是英伦媒体无孔不入的私生活轰炸。就在足总宣布最终名单前，媒体曝出的麦克拉伦绯闻还让他郁闷不已，以后这样的事情恐怕会更多！ ; ; 

　 ; 小贝的袖标保不住了？ ; 
　英足总刚刚确定麦克拉伦世界杯后继任埃里克森，英国媒体就给他来了个下马威，有人在星期四的新闻发布会上提出了关于英格兰队长的问题，麦克拉伦对贝克汉姆的躲避态度难免让人生疑。 ; ; 
　　瑞典人埃里克森在他任教期间一直让贝克汉姆出任英格兰国家队队长一职，但随着小贝在球场上表现的日益平庸，英国国内已经越来越多地出现这样的争论：是不是应该让特里或者杰拉德出任英格兰的队长？这两人在能力和领袖气质方面并不比小贝差。而本届世界杯对已经30岁的贝克汉姆来讲基本是最后一届世界杯，麦克拉伦将于8月初正式出任国家队主教练，2008年的欧锦赛是他上任后的第一届大赛，届时贝克汉姆甚至可能已经不在国家队，尽快培养新的队长也是他案头的重要任务。 ; 

　　“这些问题我现在不想讨论，世界杯就要开始了，那才是我们应该关注的事情，这个时候谈队长的问题太不恰当了。”麦克拉伦明显是在逃避媒体关于他上任后选谁出任队长的问题，即使有人提到现任队长贝克汉姆时他也没有给出积极的回应：“目前在斯文（埃里克森）手下，他依然是队长。”已经有人指出，虽然麦克拉伦曾经在国家队辅佐埃里克森5年，但他有很多自己的想法，只是因为目前还没有正式挂帅，一旦等到8月份，麦克拉伦自然会给天下人一个答案，但从目前来看，贝克汉姆的国家队队长之路似乎已经到头了。 ; 

　 ; 英格兰最大福将 ; ; 
　“最理想的主帅就是最理想的主帅，五年前是埃里克森，今天他们选择了英格兰本土教练，于是我坐在了这里，希望我在这里对英格兰本土教练是一种激励。”麦克拉伦在自己的就职演说中这样说道。从十年前刚刚入门的助理教练到足球界最具挑战性的国家队主帅，麦克拉伦不得不感叹人生大起大落实在太刺激！ ; 
　　麦克拉伦的执教生涯开始于上世纪90年代初的牛津联队，1996年他协助资深教练吉姆・史密斯执教德比郡队，使该队一举杀入英超。1999年他被弗格森招为副手，5个月后曼联在诺坎普实现三冠王伟业！2000年随着基冈的辞职，麦克拉伦和泰勒等本土教练组成了看守教练组，临时执教英格兰队，这也是此前他最接近王座的一次。2001年三月，麦克拉伦和曼联俱乐部产生了矛盾，曼联方面不愿意他继续分心给英格兰队，于是他辞去曼联助教的工作，赛季结束后他接替小罗布森成为米德尔斯堡主教练。 ; 

　　麦克拉伦在执教米堡期间，给这支中游球队带来很多变化，特别是稳定性，2004年获得了联赛杯的冠军，这是该队128年历史的第一次。去年，米德尔斯堡成绩稳中有升，获得了联赛的第七名，是他们自1951年以来的顶级联赛最好成绩，并得以历史上第二次参加欧洲赛事。在英格兰本土主帅竞争最激烈的4月份，米德尔斯堡两次在联盟杯上完成超级逆转，这也为麦克拉伦出任英格兰主帅作了最好的铺垫！ ; ; 

　 ; ; 只有米堡在叫苦 ; 
　 ; ; 麦克拉伦成为国家队主帅,这样的新闻对于米德尔斯堡俱乐部来说似乎并非什么好消息。米堡主席吉布森也在足总公布国家队主帅人选后大呼：足总选帅毁了米德尔斯堡！ ; 
　　“作为俱乐部教练，我对我的成绩感到骄傲，我已经将米堡带到了此前从未达到的高度。”麦克拉伦没有忘记令自己成名的米德尔斯堡，“现在我需要做的是带领米德尔斯堡完成这个赛季，我们还有联盟杯决赛！”麦克拉伦的感恩并没有让米堡主席吉布森舒心：“米堡有没有受到影响？当然有！选帅进行了那么长时间，没影响是不可能的！” ; 

　　不过令吉布森不满的并非麦克拉伦答应足总的邀请，而是同为候选人，温格却因为阿森纳主席邓恩在足总身居要职而受到了保护！“足总是要选一个合格的主帅，那么温格接受了足总的召见吗？惟独温格受到了保护？”吉布森的愤怒显然是针对邓恩的，昨天足总曾明确表示，温格曾经也在他们的考虑之中，足总首席执行官巴尔维克也亲口承认与温格讨论过执教英格兰队的事情。不过温格始终没有与英格兰主帅候选人联系在一起，显然邓恩为了保护阿森纳的利益没有将这个消息公布出来！ ; ;
