巨型机械大象在伦敦中心街头游行 
 ; ; ; ; ;英国伦敦5日上演了一场“巨象惊魂”。一家法国戏剧公司为了宣传新剧，斥资100万英镑制作了一头12米高、43吨重的大象和一个5.5米高的女孩模型，并通过巧妙的机械装置令它们像牵线木偶一般在伦敦中心街头游行。不料，这两个庞然大物在行走时发出的巨大响声，令当地电视台记者发生误解，以为伦敦又遭到炸弹恐怖袭击，并纷纷中断正常节目报道这一消息。而电视观众以为是恐怖袭击，大家恐慌不已。  ; ; ; ;花百万英镑制大象模型 
 ; ; ; ;据报道，制作这个大象模型的是一家名为“奢华王室”的法国戏剧公司，该公司的新剧《苏丹的大象》从7日起即将在伦敦各剧院连续上演4天，该剧讲述一名叫苏丹的发明家制造时间旅行机器，认识了一名穿越时空而来的女孩。当女孩消失后，苏丹不顾一切地想再次和她见面，并骑上大象开始跨越时空的旅行。

 ; ; ; ;每走一步就发出巨响 
 ; ; ; ;据悉，大象和女孩模型的各关节均装有先进的电动马达和液压装置，以及数十根粗钢绳，在数十名技术人员的操作牵引下，它们竟可以像牵线木偶一般在街头缓缓行走，就仿佛是有生命一般。 
 ; ; ; ;这一仿佛只有在《格列佛游记》中“大人国”才会出现的奇观，立即引来许多人的围观。然而，由于大象和女孩的体积过于庞大，当它们每行走一步，都会发出轰然巨响，甚至数英里之外都可以听到。 
 ; ; ; ;围观者造成交通堵塞 
 ; ; ; ;搞笑的是，当大象走到首相官邸唐宁街10号附近时，采访内阁改组的电视台记者听到奇怪的响声后，误以为伦敦又遭到恐怖袭击，或是有人往首相府投掷炸弹抗议。随即，电视台立即中断正常节目，抢播这一新闻。让记者如释重负的是，很快他们就明白是怎么回事，意识到虚惊了一场。 
 ; ; ; ;对于这次宣传造成的轰动效应，组织者海伦・玛里基表示非常满意。她称：“我们希望带来一些真正有纪念意义的事情，让伦敦市民欣赏。不过，也可能一些伦敦司机会感到恼火，因为围观人群造成了交通堵塞。”2 ;
