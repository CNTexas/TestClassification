　　上午8时50分左右，一名女子凄厉的呼救声打破了宁静的校园，博士楼前，一位英俊的男子被几名男博士抱着走了出来，青年男子衣服上全是斑斑血迹。 
 ; ; ; ;在他脖子上，一处伤口正朝外渗透着殷红的鲜血…… 
 ; ; ; ;青年男子，系这所重点大学的医学博士，名叫林西(化名)。知情人透露：博士刎颈自杀，竟是因为爱人的玩笑话。 
 ; ; ; ;场景一 目击 时间 9：00 
 ; ; ; ;“当时不到9点，我刚刚把锁摊子铺开。好家伙，一位男子从博士楼被抱了出来。”博士楼旁卖锁的一位大姐讲述了她目击的一幕。“太吓人了，伤者全是血。人已经昏迷了，他旁边的一个女子，哭得很伤心，不停喊救命。”卖锁大姐说，“伤者就是这个楼里的博士。我听说是在四楼用手术刀刺了自己的脖子。” 
 ; ; ; ;事发地点在校园里，根本没有出租车经过。 
 ; ; ; ;“血一直冒，我们想打120等救护车来，但肯定搞不赢。”看到男子伤情严重后，她马上丢下身边的锁摊子，跑上去帮忙拦车救人。 
 ; ; ; ;在热心大姐和其他博士的帮助下，一辆刚刚经过此处的私人轿车被拦了下来…… 
 ; ; ; ;场景二 救人 时间 9：05 
 ; ; ; ;开着车的生意人廖先生，当时刚刚经过博士楼。半小时前，他把自己的一个朋友送到医院治疗。他说，“我原本是回家，刚刚经过这里就看到有人在喊救命。”当时，廖先生的车已经远离博士楼。知道伤者情况紧急后，廖先生立即把车倒了回去。“那女子看见我停车后，竟一下跪在了我车前，喊我救她的爱人！”廖先生马上跳下车，和几位博士把伤者抬进了自己的车后排位置。 
 ; ; ; ;学校离附近的医院不远，但也要穿过几个街口。“当时街上人、车流量都非常大，我也顾不倒那么多了，挂上挡就冲。穿黄色衣服的女子，一路上紧紧把伤者抱住，把脸紧紧靠在伤者的脸上。”廖先生说，伤者的脖子一直在出血，脸上，衣服上全是血迹。 
 ; ; ; ;“你一定要挺住，不能死呀！以后，我再也不和你开玩笑了！”这名女子哭的声嘶力竭。 
 ; ; ; ;廖先生说，他从这名女子的哭诉声中，知道伤者是因为爱人的玩笑话而自杀的！ 
 ; ; ; ;“唉，不管什么原因，怎么动不动就自杀呢？他的家庭、国家把他培养成博士，容易吗？”廖先生对此感叹不已。 
 ; ; ; ;自杀博士被热心市民廖先生等送到医院后，伤者迅速被送到了急诊室抢救。经过手术清创缝合后，林博士被送到了医院住院部进一步治疗。 
 ; ; ; ;“他的伤口是住院部医生缝合的。”昨日，该院急诊室护士告诉记者，林博士脖子上的伤，不严重但也不轻。“应该说没有生命危险。”随后，记者见到了自杀的林博士。据悉，经治疗，伤者目前生命体征平稳。 
 ; ; ; ;场景三 谴责 时间：10：45 
 ; ; ; ;昨日上午10时许，记者来到了这所重点大学的博士楼。此时，博士楼前已经恢复平静。进出博士楼的博士生，依然如平常一样来往。在目击者的指点下，记者发现博士楼前的通道上果然有大量血渍散落。记者的到来，引起了博士楼管理人员的注意。很快，博士楼内一位中年妇女就拿出一个拖把，来到博士楼前，将地上的血渍一一擦干净。 
 ; ; ; ;从楼里进出的博士们，很多人对刚刚发生的一幕都非常惊讶。“不会有这样的事情在我们楼里发生吧？”在博士楼前，记者对进出博士楼的十余位博士进行了简短的采访。他们都感到非常吃惊和不理解。 
 ; ; ; ;“先不说是不是因为和爱人的玩笑而自杀，心理素质好坏值得商榷。作为一名高级知识分子，动不动就自杀，这影响肯定不好！”一位刘姓博士接受记者采访时如此表示。 
 ; ; ; ;在博士楼附近，该校一名女博士知道林博士自杀后，感到非常的惊讶。“他怎么这样漠视自己的生命呢？都博士了，这样死了的话，对得起自己的亲人吗？”这位女同学很生气的说。 ; ; ; ; 场景四 探因 时间：11：05 
 ; ; ; ;昨日，林博士的爱人陈(音)女士告诉记者，她是自贡人，这次是来成都看画展。她称，爱人林博士受伤，是因为“这段时间忙于写论文，人很疲倦没休息好，今天早上不小心摔伤的！” 
 ; ; ; ;她补充说，就是在昨天晚上，爱人写论文也写了一个通宵…… 
 ; ; ; ;陈女士断然否定了爱人是因为她的玩笑话赌气自杀的。“他就是摔伤的！” 
 ; ; ; ;但林博士的爱人所说和记者先前的采访调查结果明显不同。那么，博士到底是摔伤还是自杀？昨日，记者再次进行了深入的调查。最终在林博士的一位同事口中，证实了博士确实是自杀。 
 ; ; ; ;这位同事在记者采访时明确表示：“林博士是自杀的。自杀原因嘛，我们这些外人不好说的。”
　　・声音・生命如此脆弱 自杀代价太高 
 ; ; ; ;观点：他们内心并不像外表那样成熟 
 ; ; ; ;四川大学华西医院心理卫生中心张岚博士(心理学专家) 
 ; ; ; ;在最近5年―10年的时间里，青少年冲动性自杀行为越来越多。处于青春期的青少年，虽然身体已经是成年人，但心理还停留在不成熟的阶段，他们的内心并不像他们的外表那样成熟。在面临委屈和挫折的时候，他们选择的应对措施十分过激，往往采取伤害家人也伤害自己的行为。 
 ; ; ; ;每个人都要经历心理不成熟到成熟阶段，青少年在面对困难和委屈的时候，学会成熟理智面对。在面对压力和挫折的时候，要先缓和矛盾，再寻求解决的办法。如果你自己都不爱惜自己，别人是无法爱惜你的！ 
 ; ; ; ;珍惜生命的教育十分重要，学校和社会应该教会他们珍惜生命。处于心理转型期的青少年，他们不知道活着也是一种责任。他们想用自杀来证明自己的清白，但要知道自杀证明自己清白的代价太大了。证明清白的方式有许多种，但自杀的那种方式是最不可取的！因为你一个人的过激行为会给许多人带来伤害。在门诊中我们经常发现，自杀给家人带来的伤害非常大，家人常常因此而患上抑郁症。 
 ; ; ; ;因此，在青少年面对压力和委屈的时候，我们应该教会他们如何释放压力，比如通过体育运动、倾诉等释放压力。只有及时地释放压力，才能避免自杀悲剧的发生。 
 ; ; ; ;观点： 三大教育的缺失 
 ; ; ; ;省社科院社会学研究所副所长胡光伟教授(社会学专家) 
 ; ; ; ;胡光伟教授告诉记者，相继发生学生自杀事件很遗憾，它暴露了目前教育存在的三大缺陷。 
 ; ; ; ;一是挫折教育的缺失。从小到大，他们不缺少关爱，但缺少挫折的教育。在面临挫折的时候，哪怕是很小的挫折和委屈，自身也无法找到释放的方式，采取极端的行为来解决，这往往导致悲剧的发生。 
 ; ; ; ;二是责任感教育的缺失。家庭责任感、社会责任感教育的缺失，现在的年轻人责任心太低了，根源是极端个人主义，没有家庭责任心，没有社会责任心。诚然生命是自己的，但同时也是属于你的家庭和整个社会的。就拿那个博士生来说吧，他之所以能够走到博士这一步，是家人付出和社会培养的结果。他这样没有一点回报就自杀，显然是一种极端不负责任的做法。 
 ; ; ; ;三是珍爱生命教育的缺失。在传统的教育中，在珍惜生命教育这一环节上，还存在很多问题，这在某种程度上导致青少年不珍惜生命。在我们调查中发现，那些自杀未遂的人，事后是非常后悔的，但往往在这个时候已经晚了。作为学校，开展珍爱生命教育是很有必要的。作为一名博士生，虽然在学历上他是高智商的，但承受挫折的能力却很低。 
 ; ; ; ;其实，这些自杀的青少年，他们中的许多并不是真的想死。他们自杀，实际上是一种威胁行为。当自己的要求得不到满足的时候，他们采取这种变相的行为，迫使别人满足自己的要求。目前，处于转型时期的青少年，明辨是非的能力非常有限，不清楚什么是该做的，什么是不该做的。青少年自杀也屡见报端，可见，我们的教育确实在挫折教育、责任感教育、珍爱生命教育方面，实在是太缺乏了。 
　　・短评・ 
　　热爱生命是一种智慧 ; ; ; ; 
 ; ; ; ;前天，一位高二学生因不愿意向老师道歉而跳楼；昨天，一位博士生因为爱人开玩笑闹得不愉快而用手术刀对付了自己的脖子。在这两件事情中，我们非常触目惊心地看到了这样两个等式――“道歉”等于“生命”；“玩笑”等于“生命”。这是某些青少年心中的价值判断。 
 ; ; ; ;即使在最荒唐的魔鬼辞典上，这样的等式也是不能成立的。因为二者之间的价值和意义有天壤之别，决不可同日而语。 
 ; ; ; ;人的生死，有重于泰山和轻于鸿毛之分。一个人把自己的生命放在天平上，他所选取的对应物是什么，直接决定其价值的取向。可以肯定地说，世界上没有任何一样东西可以与生命等值，金钱不能，权力不能，任何不食人间烟火的空洞口号也不能，而玩笑和意气则更不能。 
 ; ; ; ;古代有一位隐士告诫他的弟子：凡事进三步退三步再做决定，便是智慧。弟子辞师归家，发现有人与妻同眠，怒而欲杀人烧房。突然想起临别时师傅之言，进三步，发现与妻同眠的竟是自己的母亲，退三步反思自己的莽撞，吓得大汗淋漓。 
 ; ; ; ;很多惨祸，就是在这进退的过程中产生或消解的。愿世上所有人都热爱生命，并且具备足够的保护生命安全的智慧。（曾颖）
