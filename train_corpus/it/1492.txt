 ; ; ; 由私人公司开发的新型数控飞船，将登陆月球开采新型无污染核能 
 ; ; ; ;据俄罗斯新闻网28日消息，俄罗斯“能源”火箭航天集团28日宣布，他们新发明了一种名为“TMA”的数字控制飞船。这种飞船属于“俄罗斯未来25年载人航天发展计划方案”的一部分。该飞船将被发射到月球上去提炼氦3。 
 ; ; ; ;新型数字控制系统 
 ; ; ; ;“能源”火箭航天集团曾于4月份宣布了一个“俄罗斯未来25年载人航天发展计划方案”―――――在2015年派载人飞船登陆月球执行任务，在月球上建立一个永久性基地，开发能源，并在2020年至2030年间向火星派出载人飞船。 
 ; ; ; ;目前，“能源”火箭航天集团已研制了名为“大剪刀”的宇宙飞船，以代替目前俄载人航天飞行所用的“联盟”号系列载人飞船。 
 ; ; ; ;28日，“能源”火箭航天集团宣布他们新研制了由数字控制的，可载3人的TMA飞船。“能源”火箭航天集团总裁谢瓦斯奇亚诺夫表示：“在新式登月飞船‘大剪刀’号执行任务前，我们就已经掌握了一种新的数字控制系统。这种系统可以安装在TMA飞船上，并取代其原有传统部件。” 
 ; ; ; ;到月球上勘探氦3 
 ; ; ; ;该集团将在月球上建立一个拥有核能工厂和设备、并可以提炼出氦3（helium-3）的永久性基地。他们希望这个永久基地能在2020年左右满足地球的能源需求。科学家们认为，月球上提供的氦3可以被用于未来的地球上的核聚变反应堆，能够在不产生核废料的情况下产生电力。谢瓦斯奇亚诺夫表示，月球探测计划将向地球输送10吨、价值400亿美元的氦。而TMA飞船就专门负责勘探氦3的储量，并对氦3进行提炼。 
 ; ; ; ;据悉，这一方案由私人投资，由于公司缺乏政府的支持，这一计划的可行性令人怀疑。
