
 ; ; ; 购物介绍
 ; ; ; 
汉堡是一个理想的购物天堂，这里的物价比德国其它城市都要低。在打折期时，很多品牌比香港更便宜。比较不错的购物场所有汉塞菲阿特(Hanse-Viertel)、汉堡格豪夫(Hamburger 
Hof)、鹅市(Gansemarkt-Passage)、加雷利亚(Galleria)、布莱辛霍夫(Bleichenhof)、老邮局(Alte 
Post)、考夫曼豪斯(Kaufmannshaus)、莫肯贝尔格街(Monckebergstrasse)的蕾万特商场(Levantehaus)。另外，在selmarkt周边有七家拱廊购物商场，在Gansemarkt及市政厅广场(Rathausplatz)之间有10多条名店街，最有名的是“汉诺同盟特区”(Hanse 
Viertel Arcade)，玻璃顶盖引入天然光线，明亮舒适。阿尔斯特内湖畔的Jungfernstieg也是店铺集中的地方。新墙(Neuer 
Wall)大街周边精品名店也是游客不错的选择。如果要买古董，可到古董市场(Antique Center)，地址是Klosterwall 9-21。
 ; ; ; 
在汉堡的鱼市可以在渔船买到新鲜的海鱼。现在，鱼市上除了出售鲜鱼，还有蔬菜、水果以及各种日用小商品。这就是汉堡著名的“Sunday 
Market”，每逢星期日早上，赶早市的人们都会不约而至。在这里可以体会到特殊的叫卖方式，定价后的商品包装，在里面能装多少东西就要看你的本领了。一般情况下，5马克可以买到一大包吃不完的水果。>>>汉堡美食指南
 ; ; ; 汉堡交通指南
 ; ; ; 
由于汉堡的地理环境，使得其交通非常便利。汉堡在海上对外的港口有300多个，市区横跨易北河两岸，通过易北河以及运河与欧洲内陆航远相连接。市内的交通可谓是地上和地下，国铁和私铁，公共汽车和出租车一应俱全。但游览市内最方便实用的还数自行车，这儿可以租借自行车，从几天到几个月都可以。周末骑上自行车，穿行在汉堡的大街小巷，也是别有风情。[详细] ;
